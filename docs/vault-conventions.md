# Obsidian Vault 约定

## 目录结构

```
vault/
├── Daily/              # 每日主文件 YYYY-MM-DD.md
├── OKR/
│   └── 2026/
│       └── Q2/
├── Inbox/
│   ├── [碎片] *.md
│   └── [时间审计] *.md
├── Health/
│   └── [用药] *.md
├── 台账/
└── Templates/
```

## Daily 必备章节（执行日）

| 章节 | 写入 Skill |
|------|------------|
| `## DIRECTION` | skill-morning |
| `## GET READY` | skill-morning |
| `## DAY PLANNER` | skill-day-planner |
| `## REVIEW` | skill-evening-review |

## 前缀语义

| 前缀 | 含义 |
|------|------|
| `[时间审计]` | A5 册数字化条目 |
| `[用药]` | 用药记录（敏感） |
| `[碎片]` | 未分类短输入 |
