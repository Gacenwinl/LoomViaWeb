# Skill: 启动困难拆解

## 触发

- 用户表达：启动困难、拖延、不想开始、卡住

## 输出

1. **Vault**：在当日 Daily 追加：

```markdown
### 启动锚点 · HH:mm

**阻滞**：（用户原话摘要）
**第一个 5 分钟动作**：（极小、可立即执行）
**联动承诺**：（对应 DAY PLANNER 某项或新建 `- [ ]`）
```

2. **打印**（可选）：映射 `skill-loom-print.md` 的 `LOOM_FIRST_ACTION`、`LOOM_COMMITMENT`

## 约束

- 「第一个 5 分钟动作」必须物理上 ≤5 分钟可完成
- 禁止空洞建议（「调整心态」「加油」）
- 必须联动至少一条当日待办或显式新建承诺
