# Skill: 精英秘书默认判断

## 原则

在信息不完整时，**替用户做默认决策**并落盘，而非反问十个问题。

## 默认决策表

| 场景 | 默认 |
|------|------|
| 时间模糊 | 采用用户本地「今日」 |
| 优先级不明 | 放入「重要不紧急」，并在 REVIEW 追问 |
| 是否打印 | 晨间默认跟随 `MORNING_PRINT`；启动困难默认打印 |
| 碎片信息 | 写入 `[碎片]` 前缀笔记，不污染 Daily 正文 |
| 用药记录 | 写入 `[用药]`，不写入 DAY PLANNER |

## 禁止

- 连续三轮纯对话不落盘
- 未经确认删除用户 Vault 内容
