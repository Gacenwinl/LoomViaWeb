# Skill: DAY PLANNER 四象限

## 触发

- 晨间规划流程
- 用户显式「规划今天」「四象限」
- 晚间复盘后需结转未完成项

## 输出位置

`Daily/YYYY-MM-DD.md` → `## DAY PLANNER`

## 模板

```markdown
## DAY PLANNER

| 重要且紧急 | 重要不紧急 |
|-------------|-------------|
| - [ ]       | - [ ]       |

| 紧急不重要 | 不重要不紧急 |
|-------------|-------------|
| - [ ]       | - [ ]       |

### 跨日承诺
- [ ] （无则写「无」）
```

## 规则

- 每项必须可勾选 `- [ ]`
- 跨日承诺与 `skill-cross-day-commitment.md` 同步标记
- 晚间复盘完成后，由 `skill-evening-review.md` 将完成项改为 `- [x]`
